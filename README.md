# 🤖 Offline Jarvis Voice Assistant

This project is a Python-based offline voice assistant inspired by Jarvis. It is designed to perform system-level tasks using voice commands without requiring an internet connection.

## 🚀 Features

* 🎤 Offline speech recognition (PocketSphinx)
* 🔊 Text-to-speech response (pyttsx3)
* 🖥️ Open applications (Chrome, Notepad, Calculator, Explorer)
* ⌨️ Automate typing using voice commands
* 📸 Take screenshots
* 🔉 Control system volume
* ❌ Close active windows
* 🕒 Tell time and date

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* PyAutoGUI
* PocketSphinx

## 📂 How It Works

The assistant listens to user voice input through the microphone, processes it using offline speech recognition, and executes corresponding system commands. If voice input fails, it falls back to manual text input.

## ▶️ How to Run

1. Install required libraries:

   ```bash
   pip install speechrecognition pyttsx3 pyautogui pocketsphinx
   ```

2. Run the script:

   ```bash
   python jarvis.py
   ```

## 🎯 Use Case

This assistant is useful for basic hands-free computer control and demonstrates the implementation of voice-based automation using Python.

## ⚠️ Note

* Works best in a low-noise environment
* Some commands may vary depending on system configuration
